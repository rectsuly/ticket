# ticket
A simple tickets enquiry tool written by python.

```
"""命令行火车票查看器

Usage:
    tickets [-gdtkz] <from> <to> <date>

Options:

    -d           动车
    -t           特快
    -k           快速
    -z           直达

Example:
    tickets 北京 上海 2017-10-22
    tickets -dg 成都 南京 2017-10-22
"""
```
